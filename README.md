🌐 Saloni's Connect

A full-stack web application that connects customers with trusted local service professionals such as plumbers, electricians, cleaners, and more.

---

🚀 Features

👤 Customer

- Register & Login
- Browse services by category
- Search services by area
- View worker profiles
- Book services with date & time slot
- Secure advance payment system
- View booking history
- Add reviews & ratings

👷 Worker

- Register as a service provider
- Manage availability status
- Accept / Reject bookings
- Start & complete work
- View earnings dashboard
- Edit profile

🧑‍💼 Admin

- Manage workers
- Verify service providers
- Monitor bookings
- Platform commission tracking

---

🧠 Project Flow

1. Customer registers and logs in
2. Customer selects service category
3. Chooses worker and books service
4. Makes advance payment
5. Worker accepts and completes job
6. Customer confirms and gives review
7. Admin monitors system activity

---

🛠️ Tech Stack

- Frontend: HTML, CSS, Bootstrap
- Backend: Django (Python)
- Database: MySQL
- Other: Django Templates, REST APIs

⚙️ Installation & Setup

1. Clone the repository:

git clone https://github.com/YOUR_USERNAME/salonis-connect.git
cd salonis-connect

2. Create virtual environment:

python -m venv venv
venv\Scripts\activate

3. Install dependencies:

pip install
